## Add a custom interval for sitemap preload

Edit `WPROCKETHELPERS_PRELOAD_INTERVAL_IN_SECONDS` to reflect the number of seconds you wish to apply as your preload interval. (No decimals allowed!)

To be used with:
* any setup

Last tested with:
* WP Rocket 3.0.x
* WordPress 4.9.x
